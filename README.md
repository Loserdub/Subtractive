Subtractive
Subtractive is a professional grade synthesis engine built for the modern hybrid workflow. It serves as a core module within the Hybrid Production ecosystem, providing the raw, harmonic foundations required for high level audio restoration and sound design.

This project moves away from the clutter of traditional virtual instruments. Instead, it focuses on pure oscillator stability and a refined filter architecture designed to work in tandem with the restoration tools found on jray.me.

The Vision
In a landscape crowded with algorithmic noise, Subtractive offers a return to intentionality. It is built to be the starting point for sounds that will eventually be processed, restored, and mastered through the Hybrid Production methodology. The goal is to provide a clean, predictable signal that retains its soul even after heavy digital manipulation.

Core Features
Precision Oscillators: Aliasing free waveforms designed for maximum clarity in the frequency spectrum.

Refined Filter Logic: A custom filter bridge that mimics the warmth of analog hardware while maintaining the surgical precision of digital code.

Hybrid Integration: Built to export seamless stems that are ready for the restoration and mixing pipelines used by loserdub and VISION.

Web Native: High performance audio processing optimized for modern browser environments via the Web Audio API.

Part of the Multiverse
Subtractive is not just a standalone tool. It is an entry point into a larger creative framework. It is used extensively across the various artistic identities in my portfolio:

loserdub: For crafting the harmonic foundations of lo-fi textures.

VISION: For building the weight and scale of cinematic synthesis.

Technical Implementation
This repository contains the source code for the synthesizer engine. It is written with a focus on modularity and low latency performance. Developers and artists can use this as a base for custom synthesis workflows or as a reference for implementing the Hybrid Production methodology in their own projects.

Connectivity
This project is supported by the community at r/hybridproduction. For more information on the overarching philosophy and to see the engine in action, visit jray.me.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
