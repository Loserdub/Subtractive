## 👁️ Overview: Tactile Web Audio
**Subtractive** is a high-performance, browser-based synthesizer built on React. Designed to bring the warmth and hands-on control of classic subtractive synthesis to the web, this instrument natively supports MIDI input—allowing producers to plug in their hardware controllers and play instantly with zero perceived latency.

In the era of algorithmic generation, human expression remains the vital heartbeat of music. This tool was engineered by Justin Tyler Ray as part of a broader mission to build reliable, professional-grade digital instruments that empower the "Hybrid Producer." It provides a tactile, human-driven counterweight to automated production, ensuring that the soul of the performance is never lost in the machine.

---

## 🚀 Professional-Grade Features

*   **🎛️ Subtractive Sound Engine:** A complete architecture featuring distinct oscillators, dynamic filters, and precise ADSR envelopes to sculpt classic leads, warm pads, and heavy basslines.
*   **🎹 Web MIDI Integration:** Plug-and-play compatibility with external MIDI keyboards and controllers. No drivers, no routing—just plug in and play directly in Chrome or any supported browser.
*   **⚡ Zero-Latency Performance:** Built intimately with the Web Audio API to ensure that key-presses translate to sound instantaneously, vital for capturing human groove and timing.
*   **⚛️ Reactive UI Architecture:** Constructed with React to provide a fluid, component-driven interface that responds beautifully to user input and parameter modulation.
*   **☁️ Universally Accessible:** A cloud-native instrument that bypasses the need for heavy VST installations or complex DAW routing.

## 🛠️ Technical Stack
Engineered for stability, speed, and seamless integration:
*   **Audio Core:** Web Audio API (OscillatorNodes, BiquadFilterNodes, GainNodes)
*   **Hardware Interface:** Web MIDI API
*   **Frontend Framework:** React / JavaScript (ES6+)
*   **Styling:** Modern, modular CSS designed for a distraction-free, studio-like aesthetic
*   **Deployment:** GitHub Pages

## 📥 Local Deployment & Development

To deploy the Subtractive synth locally and modify the audio engine:

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/loserdub/Subtractive.git
    ```
2.  **Navigate & Install Dependencies:**
    ```bash
    cd Subtractive
    npm install
    ```
3.  **Launch the Development Server:**
    *(Running a local server is required for Web MIDI and Web Audio API permissions).*
    ```bash
    npm start 
    # OR
    npm run dev
    ```
4.  **Connect Your Hardware:**
    Plug in your USB MIDI controller, navigate to `http://localhost:3000`, and begin playing.
